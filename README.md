# Louisville Jazz Initiative

This is the source code for the Louisville Jazz Initiative website.

**Technologies:**

- Next.js
- React
- TypeScript
- LJI Design System
- Sanity.io
- AWS Lambda
- Mailchimp
- i18n

**Getting Started:**

1. Clone the repository
2. Run `npm install`
3. Copy `env.local` from S3 into the root of the project 
4. Run `npm run dev`
